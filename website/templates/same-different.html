Demo for Stroop Task

This is a demo for the a modified version of the stroop task. I believe that in order to pursue truth in science, researchers should share experimental code when possible in order to further exploration in psychology. To that end, here is a demonstration for code that I have created for a modified version of the stroop task (actual stroop task coming soon). This code uses the jquery.js library.

Instead of indicating the color of the text, for this task you will simply be indicating whether the color of the word matches the word that is displayed, in other words whether the trials are congruent or incongruent.

In this experiment, you will see words of colors (i.e., blue or red) written in a font color displayed on the screen. Press {{}} if the word is the SAME as the color the font is in and {{}} if the font color is DIFFERENT from the color the word is displayed in. Letters are randomly assignged each time, pay close attention.
